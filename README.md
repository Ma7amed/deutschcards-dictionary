# Deutsch Cards – Offline Dictionary

This repository hosts the offline dictionary databases used by the **Deutsch Cards** Android app.

## Data Sources
- Wiktionary (via Kaikki.org dumps)
- License: **CC BY-SA 4.0**

The data has been processed, normalized, and indexed for offline use in SQLite format.

## Files
- `deutschcards_core.db` – Core A1/A2 vocabulary (shipped inside the app)
- `deutschcards_full_vX.db` – Full offline dictionary (downloadable)
